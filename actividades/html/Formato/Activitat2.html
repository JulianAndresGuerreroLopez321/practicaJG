Bienvenidos a Nuestra Página

Esta es una página de ejemplo creada para mostrar el uso correcto de encabezados y párrafos en HTML. Su estructura es sencilla y clara, ideal para proyectos básicos o para quienes están aprendiendo desarrollo web.

Sobre el contenido

El contenido está organizado de forma semántica, utilizando etiquetas HTML que ayudan tanto a los usuarios como a los motores de búsqueda a entender mejor la información presentada.

Nuestro objetivo

El objetivo de esta página es servir como referencia simple y práctica. A partir de esta base, se pueden agregar estilos CSS, scripts o más secciones según las necesidades del proyecto.
